# Supply Run Pressure Table

*Temporal pressure for the secondary pass. The party is outside the walls for the first time. Time costs resources: the siege is ongoing, Maret's window is counting down, and Aldric has been out here for two days. Roll once per scene transition (d6).*

## Pressure Table (d6)

| Roll | Event |
|---|---|
| 1–2 | **Quiet.** The pass is still. No patrol, no movement. The silence is not reassuring — it means nobody is watching because they don't need to. |
| 3 | **Ridge patrol.** A Reconstitution archer on the northeast ridge tracks the party's movement. No attack — the party is in supply-run context, not military threat. But the patrol notes them. The Reconstitution now knows the party left the walls and returned with the cart. |
| 4 | **Time pressure.** Sava counts the hours. If the party has spent more than 90 minutes outside the walls, Maret's six-day window is now visibly close. Sava makes a DC 10 Medicine check: she calculates exactly how many days Maret has left. The answer is 4 if the cart makes it back; 0 if it doesn't. |
| 5 | **Second tracks.** Talis notices a third set of boot prints that diverges from the ambushers' trail — someone left the choke point heading east, not south. The third person from the upper ledge? Or a scout watching the watchers? This is not a combat event; it is a question the session log should file. |
| 6 | **Aldric stirs (if not yet found).** If the party has not reached Room 03, a faint sound carries up the trail — a groan, irregular. DC 12 Perception to hear it and identify the direction (north end of the choke, Room 03). This accelerates the party toward the choke without mandating combat entry. |

## Night/Return Pressure

On the return trip (Room 05 → Room 01), roll once. The same table applies. A result of 3 (ridge patrol) on the return means the Reconstitution watched the cart come back — which is noted for the Reconstitution's intelligence picture.
