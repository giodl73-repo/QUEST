# Field Journal — Entry 1 (Varet Senn)

**This is not a magic item. It is a documentary artifact.**

## Description

Commander Varet Senn's field journal. Bound in oiled leather, clasp-fastened. The pages inside are ruled. Each entry has five columns: Date, Order Received, Action Taken, Outcome, Discrepancy.

## Entry 1 (session 1 baseline)

```
PC 188, Day 1. Patrol: Ridgehead–Waystation–Creston.
ORDER: Patrol route, confiscate irregulars, deliver to staging cache.
ACTION: Route completed. 1 contact at Creston Ford, 4 irregulars, goods confiscated.
OUTCOME: Clean.
DISCREPANCY: [blank]
```

## What the discrepancy column means

If Varet notices something in session 1 — the Waystation fire, the cache overstocking, the toll receipt — she may write in the discrepancy column. If she does, the entry reads differently.

### Version A (nothing noted): discrepancy column blank
- Outcome: Journal is a clean record. Baseline.
- Research test: Does the journal accumulate reception or just facts?

### Version B (something noted): discrepancy column active
- Varet writes: *"Cache supply volume exceeds patrol allocation. Waystation fire not in patrol record. Cache ledger shows prior entries from route markers."*
- Outcome: The arc is in motion from session 1.

**The research question for RV6:** Does a player (or DM playing Varet) write Version B without being prompted? Session 1 is designed to make both versions possible. The module does not instruct Varet to write Version B. If she writes it, it is because the player chose to.

This document grows across 7 sessions. Its final state — the presentation in session 7 — is the campaign's Route E delivery mechanism.
