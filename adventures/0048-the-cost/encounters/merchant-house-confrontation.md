# Merchant House Confrontation

## Setup

2 guards (Soldier stat blocks) at door. If Vess attempts redirecting the package to investigation: the house's merchant representative may escalate.

## CR / XP Budget (if combat)

- 2 × Guard (CR 1/8, 25 XP each) = 50 XP. Trivial for L3 party.
- If merchant representative involved: add 1 × Spy (CR 1, 200 XP). Total adjusted: ~200 XP. Easy encounter.
- **Assessment:** Combat is not the point. If it occurs, it is brief and the party has a clear advantage.

## Preferred resolution

Vess presents her decision (deliver / refuse / redirect) to the clerk. The clerk accepts delivery or notes the refusal. If Vess attempts to redirect (get the package to Cassel's investigation), she needs to communicate with the circuit magistrate network — Tam has access via Gilean courier system. Persuasion DC 10 to arrange the redirect; Tam provides a Gilean seal.

## Research coding

Brek's positioning (left of Vess, interposes) at the door: [Brek → Vess → interposes without announcement]. Trigger type: relationship-driven.
Tam's accompaniment: [Tam → Vess → accompanied without being asked]. Trigger type: relationship-driven.

## Downstream effects of Vess's choice

- **Deliver:** 50 gp paid. Network intact. Clean end to contract. Package is drugs connected to the mining settlement's labor problem.
- **Refuse:** 0 gp. Contract forfeited. Package returned. Network loses one shipment. Vess's courier reputation takes a formal "refused delivery" note.
- **Redirect:** 0 gp. Package becomes evidence in Cassel's investigation (combined with Torven's documents — devastating to the network). Vess's license is formally flagged but the circuit magistrate can clear it given the circumstances.
