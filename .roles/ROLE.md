# QUEST Role Index

QUEST is a D&D adventure workshop with design skills, playtest rubrics,
emergent player styles, and a deterministic session engine. Use these roles
when changing adventure design, playtest scoring, player-style promotion,
session state, or public methodology claims.

## Parliament

Governance roles decide whether a session, rubric, or release is coherent and
safe to advance.

| File | Role | Primary tension |
|---|---|---|
| `parliament/rubric-governance-steward.md` | Rubric Governance Steward | Learning from play vs. retroactive score drift |
| `parliament/session-integrity-auditor.md` | Session Integrity Auditor | Narrative flow vs. complete seven-stage evidence |

## Adventure craft

Adventure craft roles protect the table experience and scenario design.

| File | Role | Primary tension |
|---|---|---|
| `adventure-craft/treasure-story-curator.md` | Treasure Story Curator | Meaningful loot decisions vs. decorative rewards |
| `adventure-craft/table-module-editor.md` | Table Module Editor | Rich scenario texture vs. runnable DM-facing module |

## Playtest

Playtest roles protect observed player behavior and emergent styles.

| File | Role | Primary tension |
|---|---|---|
| `playtest/player-style-observer.md` | Player Style Observer | Emergent catalog growth vs. premature taxonomy |

## Engine

Engine roles protect deterministic mechanical support for live play.

| File | Role | Primary tension |
|---|---|---|
| `engine/dice-state-auditor.md` | Dice State Auditor | LLM narrative freedom vs. reproducible mechanical state |

## Review order

1. Use Adventure craft roles while designing adventures, treasures, NPCs, and modules.
2. Use Engine roles for dice, state, loader, and event-log changes.
3. Use Playtest roles after sessions and before promoting player styles.
4. Use Parliament roles before rubric bumps, handoffs, or public claims.
